# Classifying-Types-of-Street-Waste-using-Deep-Learning
Accurate classification of street waste into distinct categories such as plastic, metal, glass, organic, and paper remains a significant challenge for urban waste management due to the rapid increase in waste generation, labor‑intensive manual sorting, and high variability in waste appearance. 
This commit introduces the implementation of a deep learning–based pipeline for the classification of street waste images as part of the project titled “Classifying Types of Street Waste using Deep Learning.” The main objective of this update is to develop and train a robust image classification model capable of identifying different categories of waste commonly found in urban street environments. The system is designed to support automated waste monitoring and smart waste management solutions by accurately detecting and classifying waste materials from images captured in real-world conditions.

The first step included preparing the dataset required for model training and evaluation. The waste classification dataset was obtained from publicly available sources and organized into appropriate directory structures for training, validation, and testing. Data preprocessing was performed to ensure that the images were compatible with deep learning models. Each image was resized to a fixed resolution to maintain consistency across the dataset. Normalization was applied to scale pixel values and improve model convergence during training. The dataset was further divided into training and validation sets using an automated splitting approach to ensure that the model could be evaluated on unseen data during training.

To improve the generalization ability of the model, several data augmentation techniques were implemented. These augmentations help simulate real-world variations that may appear in street waste images, such as changes in orientation, lighting conditions, and camera angles. Random horizontal flipping, random rotations, and basic geometric transformations were applied to the training images. These techniques effectively increase dataset diversity without collecting additional images and help reduce the risk of overfitting during model training.

After completing the preprocessing and augmentation stages, the next step involved implementing the deep learning architecture for waste classification. In this update, a transformer-based approach was explored by integrating the DINOv2 backbone for feature extraction. DINOv2 is a self-supervised vision transformer model that is capable of learning powerful visual representations from images. By leveraging the pretrained DINOv2 model, the system can extract high-quality visual features from waste images even when the amount of labeled data is limited. This approach allows the model to capture complex visual patterns and semantic relationships that may not be easily learned by traditional convolutional neural networks.

On top of the DINOv2 backbone, a custom classification head was designed to perform the final prediction task. The classification head consists of multiple fully connected layers combined with batch normalization, GELU activation functions, and dropout regularization. The purpose of these layers is to transform the extracted feature embeddings into class probabilities corresponding to the waste categories. Batch normalization helps stabilize training and accelerate convergence, while dropout reduces the possibility of overfitting by randomly deactivating neurons during training.

The model training pipeline was implemented using a modular deep learning framework that supports scalable experimentation and simplified training management. The training process was configured using a GPU-enabled environment to accelerate computation and enable efficient processing of large batches of images. Mixed precision training was also enabled to reduce memory consumption and improve training speed while maintaining numerical stability. During training, the AdamW optimizer was used to update model parameters, and a cosine annealing learning rate scheduler was applied to gradually adjust the learning rate across training epochs.

The training process was conducted for multiple epochs while monitoring both training loss and validation performance. Cross-entropy loss was used as the objective function since the problem is a multi-class classification task. During each training iteration, the model predicted the class labels for the input images and compared them with the ground truth labels to compute the loss value. The optimizer then adjusted the model weights through backpropagation in order to minimize this loss. Validation metrics were recorded at the end of each epoch to track how well the model generalized to unseen data.

After completing the training phase, the model was evaluated on a separate test dataset to measure its final performance. Evaluation metrics included classification accuracy and loss values. The results showed that the trained model achieved approximately 96 percent test accuracy on the waste classification dataset. This indicates that the transformer-based architecture is capable of effectively distinguishing between different types of waste materials. The strong performance demonstrates the potential of advanced vision transformer models for real-world environmental monitoring tasks.

In addition to the model implementation, this commit also includes the complete training pipeline, dataset handling procedures, preprocessing scripts, and evaluation workflow within the project notebook. The notebook provides a reproducible environment where the entire training and evaluation process can be executed step by step. This ensures that the experimental results can be replicated and extended in future development phases.

This update represents a significant milestone in the development phase of the project. It establishes a functional deep learning pipeline that can process waste images, extract meaningful visual features, and generate accurate classification predictions. The successful integration of transformer-based models provides a strong foundation for further experimentation and comparison with other architectures such as convolutional neural networks and alternative vision transformers.

Future work will focus on expanding the model evaluation process and comparing multiple architectures to determine the most effective solution for street waste classification. Planned improvements include implementing additional transformer models, exploring segmentation-based preprocessing methods to reduce background noise, and performing detailed performance analysis using metrics such as precision, recall, F1-score, and confusion matrices. These enhancements will help refine the system and improve its reliability when deployed in real-world waste monitoring applications.

Overall, this commit documents the successful development and training of a transformer-based waste classification model along with the associated data processing pipeline, training configuration, and evaluation results. The implemented system demonstrates strong performance and forms the basis for the remaining stages of the project, including advanced model experimentation, comparative analysis, and final system integration.
